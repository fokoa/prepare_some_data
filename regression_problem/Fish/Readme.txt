=========================================
Content
=========================================

This dataset is a record of 7 common different fish species in fish market sales. With this dataset, a predictive model can be performed using machine friendly data and estimate the weight of fish can be predicted.


=========================================
Files
=========================================

	- fish.csv
	

=========================================
Attribute Information
=========================================	
	
	0 Species : species name of fish
	1 Weight : weight of fish in Gram g
	2 Length1 : vertical length in cm
	3 Length2 : diagonal length in cm
	4 Length3 : cross length in cm
	5 Height : height in cm
	6 Width : diagonal width in cm
	

=========================================
Inspiration
=========================================

Multiple linear regression is a fundamental practice for this dataset. Multivariate analysis can also be performed.
